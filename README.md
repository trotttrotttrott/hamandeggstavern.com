# Ham and Eggs

#### Getting started

1. Clone the repo
```
git clone git@github.com:trotttrotttrott/ham_and_eggs.git
```

2. Create and update .env file
```
cp .env.sample .env
```

3. Install bundle
```
bundle install
```

4. Start the server
```
forman start
```

5. Go to http://0.0.0.0:5000

#### Running tests

```
bx rake
```
